# 谜境拼图微信小程序

原生微信小程序 MVP，按“剧情包装拼图”思路搭建，并额外支持玩家自定义图片关卡。

## 当前已完成

- 首页 -> 系统章节 -> 关卡简报 -> 拼图 -> 结算 -> 个人中心
- 3 个系统章节、12 个示例关卡
- 本地存档、体力恢复、签到、金币、激励广告占位
- 拼图交换、提示、重置、失败复活、结算奖励
- 自定义拼图中心
- 从相册选图生成关卡
- 不同切片规格：`3x3`、`4x4`、`5x5`、`3x4`、`4x5`
- 挑战码复制、导入、保存与再次游玩

## 当前分享方式

- 已支持：`挑战码分享`
  - 生成关卡后复制挑战码发给朋友
  - 对方在自定义中心粘贴导入后即可游玩
- 还未支持：`一键转发后直接打开同一张图`
  - 这类分享需要云存储或后端，把图片上传后生成可公开访问的关卡 ID
