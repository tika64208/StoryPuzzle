# 谜境拼图 提审文案

## 版本说明

本次版本为《谜境拼图》剧情版迭代更新。游戏以剧情拼图为核心，玩家通过拖拽碎片逐步拼回完整画面，并在每一关解锁对应的故事线索。当前版本已上线完整的《倩女幽魂·双生局》十六回主线章节，支持章节旁白、关卡开场剧情、通关后原图欣赏与剧情揭晓。

本版本新增章节难度选择，玩家可按章节选择 `3 星 / 4 星 / 5 星` 三档挑战，对应 `3x3 / 4x4 / 5x5` 碎片规格；同时新增挑战得分与微信好友排行，便于玩家在完成剧情闯关后继续进行难度挑战与好友比较。版本内同时包含提示、定格符、引路符、失败复活、签到补给、设置系统、隐私摘要、用户协议和发布检查等基础能力。

此外，游戏支持自定义谜境玩法，玩家可以从相册上传图片或使用默认样图生成拼图关卡，并通过谜境码分享给好友体验。

## 一句话简介

一边拼回《倩女幽魂》十六回旧梦，一边自由选择挑战难度，也能把自己的照片做成可分享的谜境关卡。

## 功能亮点

- 完整主线章节：《倩女幽魂·双生局》16 关剧情拼图
- 章节旁白、单关开场、通关故事揭晓三段式叙事
- 章节难度可选：`3 星 / 4 星 / 5 星`，对应 `3x3 / 4x4 / 5x5`
- 挑战得分累计与微信好友排行展示
- 拖拽拼图、拼块组合移动、吸附反馈与通关庆祝效果
- 自定义照片拼图、默认样图生成、谜境码分享
- 提示、定格符、引路符、失败复活、签到补给、设置系统

## 适合平台提审的简短更新说明

- 新增章节难度选择，支持 `3 星 / 4 星 / 5 星` 三档挑战
- 新增挑战得分系统与微信好友排行展示
- 新增独立设置系统，支持音效和震动开关
- 优化章节进入流程与通关结果展示
- 保持《倩女幽魂》16 关主线剧情与自定义谜境玩法

## 审核关注点说明

- 游戏为单人休闲拼图玩法，无陌生人语音、文字聊天或匹配对战系统
- 排行功能仅基于微信好友关系展示挑战分，不提供陌生人社交能力
- 当前未开通内购，主要为非付费体验
- 自定义图片默认仅保存在本地，玩家主动复制谜境码时才会进行分享
