# 谜境拼图 更新日志

## 2026-04-06

### 新增

- 新增章节难度系统：每个章节支持 `3 星 / 4 星 / 5 星` 三档挑战。
- 新增难度映射规则：分别对应 `3x3 / 4x4 / 5x5` 碎片规格。
- 新增玩家主动选难度能力：可在章节页和章节开场弹层中直接切换难度。
- 新增挑战得分系统：挑战 `3 星` 记 `3` 分、`4 星` 记 `4` 分、`5 星` 记 `5` 分。
- 新增总挑战分累计与结算展示。
- 新增微信好友排行页，按挑战分高低展示好友排名。
- 新增首页 `好友排行` 入口。
- 新增首页 `设置系统` 按钮与常规齿轮图标入口。
- 新增独立设置页，支持音效与震动开关。

### 优化

- 章节难度按章节保存到本地存档，重新进入时自动沿用上次选择。
- 同一关卡只记录历史最高挑战分，避免重复刷分累计。
- 首页主卡补充挑战分展示，结算页补充挑战得分与总挑战分展示。
- 将章节完成状态文案中的“星数”调整为“评级”，避免与难度星级混淆。
- 好友排行在无法拉取真实数据时提供降级提示，便于开发阶段调试。

### 修复

- 修复章节列表绘制时 `selectedDifficulty is not defined` 导致的页面报错。
- 修复设置页入口在真实小游戏运行链路中缺失的问题。
- 修复震动开关未正确影响震动反馈的问题。

### 文档

- 更新 `README.md`，同步当前运行链路、功能清单与工程结构说明。
- 更新 `SUBMISSION_NOTES.md`，补充难度、排行、设置系统的提审表述。

## 2026-04-05 首发版

### 本次上线内容

- 微信小游戏版主流程已完成：首页、章节页、关卡页、拼图页、结算与补给站可正常联通。
- 系统剧情重构为单章 `倩女幽魂`，共 `16` 关，按完整故事线推进。
- 系统剧情图已替换为 `16` 张新素材，资源目录为 `assets/story/qiannv-youhun16/`。
- 小游戏端已支持章节旁白、单关开场剧情、通关后原图欣赏、通关后剧情揭晓。
- 拼图玩法已支持拖拽碎片、拼块组联动拖拽、吸附动画、交换反馈与通关粒子效果。
- 道具系统已接入：`提示`、`定格符`、`引路符`、失败复活。
- 自定义谜境已支持默认样图、相册选图、谜境生成、谜境码导入导出。
- 音效与氛围音已接入，包含拼合、引路、定格、失败、成功等反馈。
- 上线基础能力已具备：隐私摘要、用户协议、发布检查、运行日志导出。

### 本次重点优化

- 首页、章节页、关卡页文案统一为 `倩女幽魂十六回` 的剧情口吻。
- 删除旧的 `assets/story/qiannv-youhun/` 7 张素材，避免资源混用。
- 修复小游戏资源路径兼容问题，系统剧情图优先走小游戏稳定路径。
- 修复单块与单块交换时误带动其他碎片的问题。
- 优化通关节奏，不再直接跳出，而是先欣赏图片，再展示剧情。

### 当前版本定位

- 这是可提审的首发剧情版。
- 当前核心卖点是 `剧情拼图 + 国风志怪 + 自定义照片谜境`。
